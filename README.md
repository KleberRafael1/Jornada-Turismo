# 🌍 Jornada Turismo

Projeto desenvolvido como desafio final da **Fase 3 do programa Oracle Next Education (ONE)** em parceria com a Alura.

O objetivo foi desenvolver um **site completo para uma agência de turismo**, com design moderno, navegação intuitiva e layout responsivo.

## 🚀 Deploy

Acesse o projeto online:

https://jornada-turismo.vercel.app

## 💻 Sobre o projeto

O **Jornada Turismo** simula o site institucional de uma agência de viagens.

Diferente de uma landing page simples, o projeto foi desenvolvido como um **site completo**, com múltiplas seções e organização de conteúdo semelhante a projetos comerciais.

## 🎯 Funcionalidades

- Página inicial com apresentação da agência
- Seção de destinos turísticos
- Apresentação de pacotes de viagem
- Layout moderno e organizado
- Navegação intuitiva
- Design responsivo para diferentes dispositivos

## 🛠️ Tecnologias utilizadas

- HTML
- CSS
- Flexbox
- CSS Grid
- JavaScript
- Vercel (Deploy)

## 📱 Responsividade

O site foi desenvolvido para funcionar em:

- Desktop
- Tablet
- Smartphone
